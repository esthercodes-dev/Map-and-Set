This code introduces ES6 Map and Set structures for storing key/value data, removing duplicates, tracking word frequency, creating a phonebook, and performing set operations like union, intersection, and difference. Map helps manage structured data, while Set ensures uniqueness in collections.

How to Run

Place all examples into a single .js file (e.g., index.js). Run the file with Node.js using the command node index.js, or copy each snippet into your browser console to see the results printed immediately.
